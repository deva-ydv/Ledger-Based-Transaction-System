Ledger-Based Transaction System

● Server Status: RUNNING
Description:

Secure backend ledger system with MongoDB ACID transactions, idempotency, and double-entry accounting.

Features:
  • Ledger-based balance calculation
  • Double-entry accounting (DEBIT/CREDIT)
  • MongoDB ACID transactions
  • Idempotent transaction handling
  • JWT Authentication
  • Email notifications
API Endpoints:
  • POST /api/auth/signup
  • POST /api/auth/login
  • POST /api/auth/logout
  • POST /api/transaction
  • POST api/system/initial-funds
  • GET /api/account/balance
  • GET /api/accounts
